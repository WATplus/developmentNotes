## 实体字符串

- 有的字符是没有办法在 HTML 文档中打出来的 
   - 可能导致程序出现异常 
   - 也可能无法显示
- 语法`&***;`
- **常见的实体字符串:**
   - `&nbsp;` => `空格`
   - `&lt;` => `<`
   - `&gt;` => `>`
