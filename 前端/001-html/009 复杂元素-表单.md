# 表单

## 常见表单标签

- `<form>`标签的根元素
   - `action = ""`服务器地址
   - `method = "" ` 请求类型`get/post`
   - `target=""` 响应显示的位置 `_blank`会在现页面显示相应
- `input` 元素 获取用户输入
- `label`用与将内容与表单元素绑定
- `select/opation`下拉菜单元素
- `textarea` 多行文本域

## input 元素
### 常见input元素类型

- `type=""`
   - text
   - password
   - time
   - date
   - color
   - submit
   - reset
   - button
   - ...

### 常用属性

- `disable`禁用
- `autofoucs`自动获得焦点
- `readonly`只读
