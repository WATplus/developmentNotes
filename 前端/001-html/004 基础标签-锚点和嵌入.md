# 基础标签-锚点和嵌入

## anchor 锚点标签

- `<a></a>`
- 创建一个**锚点 **
- **主要属性**
   - `href = ""` 跳转的目标
   - `**target = ""**`** 目标页面的打开方式**
      - `**_self**`**当前标签页**
      - `**_blank**`**空白页**
      - `_parent` 和 `_top` 配合 ifram 使用

## iframe 嵌入网页

- `<firame>` 
- **主要属性**
   - `**src**`** : **被嵌入的网页的地址
   - `frameborder` : 框架是否有边框 , 取值 0 或 1
- **框架中的链接的**`**target**`
   - **因为**`**iframe**`**所以页面之间可能存在嵌套关系**
   - `_parent`就是在父级中打开
   - `_top`就是在最顶层页面中打开
